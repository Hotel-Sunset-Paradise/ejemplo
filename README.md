## Hotel Sunset Paradise ##
 El Sistema de Gestión Hotelera se plantea como una solución tecnológica para optimizar los procesos administrativos y operativos del hotel. Actualmente, la gestión manual de reservas, pagos y control de personal genera sobreventas, errores en la facturación y dificultades en la organización.
